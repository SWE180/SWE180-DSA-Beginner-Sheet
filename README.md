# 🚀 SWE180 Beginner DSA Sheet

The **SWE180 Beginner DSA Sheet** is a curated list of **141 must-do problems** covering all the fundamentals of Data Structures & Algorithms:
✅ Arrays, Linked Lists, Stacks, Queues
✅ Trees, Graphs, Tries
✅ Recursion, Dynamic Programming, Backtracking
✅ Searching, Sorting, Bit Manipulation
✅ Hashing, Matrices, Strings, and more!

Our goal is simple: **make DSA learning structured, beginner-friendly, and placement-focused.**
